# Project Context: 砂職人の店 (Su-Na Pot)

## 概要
愛知県の鋳物工場から排出される廃砂（産業廃棄物）をアップサイクルした植木鉢「Su-Na（スーナ）ポット」の製造・販売プロジェクト。
「砂職人の店」というブランド名で展開。

## Value Proposition (提供価値)
1.  **植物を美しく見せる「黒」の背景**
    *   マットな黒色が光を吸収し、植物のグリーンの輪郭を強調する。
    *   葉の色が深く、鮮明に見える（美術館のライトアップ効果）。
2.  **独特の質感**
    *   鋳型に使われた砂ならではの「ザラザラとした」質感。
    *   無塗装で、砂の粒子が重なり合った「奥行きのある黒」。
    *   無骨さと、職人の手仕事による丸み（温かみ）の共存。
3.  **Sustainability (SDGs)**
    *   本来捨てられるはずの産業廃棄物（廃砂）を再利用。
    *   「おしゃれだから選んだ選択が、実は地球にも優しい」という心理的充足感。
4.  **Affordability**
    *   数百円〜数千円という手に取りやすい価格帯。
    *   「心地よい選択」を多くの人に届けるため。

## Brand Identity
*   **Tone & Manner**: シック、モダン、静か、温かみ、誠実、誇り。
*   **Keywords**: アップサイクル、鋳物砂、黒の魔法、背景色、ストーリー、経年変化、産業廃棄物、職人の手仕事。

## Related Products
*   **デコジャリ**:
    *   三州瓦の廃材をリサイクルした化粧砂利。
    *   黒いスーナポットに白いデコジャリを合わせることで、高級感（セレクトショップのような雰囲気）を演出する。

## Target Audience
*   観葉植物をインテリアに取り入れているが、「なんとなくパッとしない」と感じている層。
*   モノの背後にある「ストーリー」や「意味」を重視する層。
*   環境配慮（SDGs）に関心があるが、無理なくオシャレに取り入れたい層。
