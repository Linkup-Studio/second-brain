# 第15話：副業を続けられたのは、“嫌だった”から。

こんにちは、ぐっさんです✨
会社員をしながら、副業で公式LINEの構築をしていまーす。
さて、今日は**「どうして副業を続けられているのか？」**というテーマでお話ししてみたいと思います。

### “普通のサラリーマン”が嫌だった

副業を始めた理由を聞かれると、やっぱりここに戻ってきます。
「ただ、普通のサラリーマンで終わりたくなかった」
それが、私の最初の原動力でした。
日々、会社と家を往復して、決まった給料をもらいながら過ぎていく日常。
その中でふと、「このままでいいのか？」って思うようになったんです。
「働きたくない」わけじゃない。
でも、<strong>“会社に行かなくてもいい状態”を自分の手でつくれるようになりたい</strong>。
そう思うようになりました。

### 続かなかった副業たち

そこから、いろんな副業に挑戦しました。
ブログ、せどり、アフィリエイト、SNS運用、個性診断。
どれも「いいな」と思って始めたけど…どれも続きませんでした。
やってるうちに、「これ、ほんとに自分に合ってるのかな…」って思ってしまって。
熱が入らなくて、手が止まって。気づけば、やめてしまう。
“理想”はあったけど、“現実”はなかなか変わらなかった。

### それでも、やめなかった

でも、「あきらめる」という選択肢はありませんでした。
なぜかというと、<strong>“会社に縛られたくない気持ち”が強すぎた</strong>から。
「このままで終わるのは絶対イヤだ」っていう想いだけで、なんとか走り続けてた。
たぶん、“夢に向かって”じゃなくて、“今が嫌だった”から、やめられなかったんだと思います。

### 初めて、自然と続けられた副業

そんな中で出会ったのが、公式LINE構築。
「これもダメだったら、どうしよう…」と思いながら、
とりあえずやってみたら、気づけば手が動いていた。
設定やデザインを考えてる時間が、すごく楽しくて。
さらに、<strong>AIを使って文章や動画をつくることにもハマっていった</strong>。
“無理してやる”じゃなく、“もっとやりたい”になってた。

### モチベーションが落ちない、という不思議

そして、びっくりしたのは、
この10ヶ月間、<strong>一度もモチベーションが落ちたことがない</strong>ってこと。
正直、自分でも不思議です。
でもたぶん、楽しいだけじゃなく、「これは自分の武器になる」と感じているからだと思います。

### 副業を続けてよかったと思う瞬間

noteを書いたり、Xで発信したり。
誰かに見てもらえる場所ができたこと。
そして、「やってるよ」と伝えられるものがあること。
それだけで、「続けてきてよかったな」と思えるようになりました。

### 最初は、“イヤだったから”でいい

副業って、“やりたいこと”がある人だけのものじゃないと思います。
私みたいに、
「このまま終わりたくない」とか、「今を変えたい」とか、
そんな“モヤモヤ”から始めてもいい。
むしろ、その気持ちがある人にこそ、
「自分に合うもの、ちゃんとあるよ」って伝えたいです。

ここまで読んでくださって、ありがとうございました。
ぐっさんでした！
