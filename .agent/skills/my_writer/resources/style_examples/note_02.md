# 第8話：LINE構築は“伝え方の工夫”で変わる。AIを使い始めて気づいたこと

こんにちは、ひろのり✨です
会社員をしながら、副業で公式LINEの構築をしていまーす。
さて、今日は**「AIを使った伝え方の工夫」**というテーマでお話ししたいと思います。

公式LINE構築を始めて10ヶ月。
まだまだ手探りな部分はありますが、少しずつ“自分らしいスタイル”が見えてきました。
その中でも、私が特に大切にしているのが、**「伝え方」**です。
どうすれば、もっと伝わるか？もっと伝えやすいか？
最近は、AIの力も借りながら、そんな工夫を重ねています。

### AIって、ただの効率化じゃない

最初にAIを取り入れたのは、モニター案件だったみっちょさんの構築のときでした。
「あいさつ文、どうしようかな…」と悩んでいたときに、
ふと、AIに案を出してもらったら意外と良い文章が出てきて。
それ以来、<strong>あいさつ文や自己紹介文は、AIで文章を生成して、そこから自分らしく整える</strong>というスタイルが自然と定着してきました。

### 「文字だけじゃ伝わらない」って気づいた

LINEってとても便利だけど、“文字だけ”だと伝わりきらないこともあるんです。

*   声のトーン
*   表情
*   雰囲気

こういう“空気感”があるだけで、相手の安心感が全然違う。
そこで最近は、<strong>よくある質問の回答にAI動画を活用</strong>するようになりました。
たとえば、 「Q. 体験申込はどうすればいいですか？」
という質問に、ただテキストで返すのではなく、
<strong>動画で伝えることで、より親しみやすく伝わるように</strong>しています。

### 「伝え方」を学んだ原点は、個性診断師のとき

実はこの「伝え方」へのこだわりは、
以前やっていた<strong>個性診断師の副業</strong>から来ています。
その人の診断結果を伝えるとき、専門用語ばかりでは意味がない。
いかにその人に合わせて、わかりやすく、丁寧に伝えるか。
そんなことを何度も考えてきた経験が、
今のLINE構築にも生きていると感じています。

*   自分の言葉で伝える
*   難しいことを、やさしく伝える
*   相手の“受け取りやすさ”を大事にする

その延長線上に、AIという選択肢がある。
私はそう思っています。

### セミナーで伝える側になって感じたこと

最近では、友人と一緒に<strong>公式LINE構築セミナー</strong>も開催しました。
友人がLINE全体の使い方を説明し、
私は<strong>AIで文章をつくる方法</strong>について担当。
「こんな方法あるんだ！」
「思ったより簡単かも！」
そんな反応をいただいて、「伝えるって、やっぱり楽しいな」と改めて感じました。
自分にとっては当たり前の工夫でも、
人にとっては価値ある情報になる。
それを実感できた、すごく大きな出来事でした。

ここまで読んでくださって、ありがとうございました。

次回は、公式LINEを導入したことで「現場がどうラクになったか？」というリアルなお話です。
リフォーム会社／OB会／美容室…それぞれの変化を感じてもらえたら嬉しいです！
