

## 【1週目】まずは知ってもらう・興味を惹く

- [ ] **Day 1：色の魔法**
    
    - **内容**: 植物が一番きれいに見える色って知ってますか？答えは「黒」。背景が黒だと、グリーンの鮮やかさが驚くほど引き立つんです。
        
    - **📸写真**: スーナポットの黒と鮮やかなグリーンのコントラスト
        
- [ ] **Day 2：水やりの極意**
    
    - **内容**: 「枯らしてしまう」原因の1位は水のやりすぎ。土が乾くまでグッと我慢。あげる時は底から出るまでたっぷりと。このメリハリが大事🌿
        
    - **📸写真**: ジョウロで水やりしている手元
        
- [ ] **Day 3：素材の正体**
    
    - **内容**: この植木鉢、実は「砂」でできています。鋳物工場で捨てられるはずだった砂を再利用。ザラッとした独特の質感が、部屋の空気を変えてくれます。
        
    - **📸写真**: 鉢の表面のアップ（ザラザラ感）
        
- [ ] **Day 4：植物の着替え**
    
    - **内容**: プラスチックの鉢から着替えるだけで、植物は一気に「インテリア」になります。服を選ぶように、鉢も選んでみませんか？
        
    - **📸写真**: プラ鉢とスーナポットの並び比較
        
- [ ] **Day 5：【植物のケア】**
    
    - **内容**: 葉っぱに元気がない時は、霧吹きで「葉水」を。乾燥を防いで、害虫予防にもなりますよ💧
        
    - **📸写真**: 霧吹きで水をかけているキラキラした写真
        
- [ ] **Day 6：職人の温もり**
    
    - **内容**: 職人の手仕事が生む、絶妙な丸み。無骨な黒い砂なのに、どこか温かい。大量生産にはない表情を楽しんでください。
        
    - **📸写真**: 職人が作業している風景 or 鉢のシルエット
        
- [ ] **Day 7：日曜日のルーティン**
    
    - **内容**: 日曜日は、お気に入りの植物を窓辺に移動させて日光浴。スーナポットなら、移動も少し楽しくなります。よい休日を☕
        
    - **📸写真**: 窓辺に置かれた植物とコーヒー

## 【2週目】深掘りとメリット

- [ ] **Day 8：仕上げの魔法**
    
    - **内容**: 土が見えているのが気になるなら「デコジャリ」を。土隠しをするだけで、清潔感も高級感も段違いにアップします✨
        
    - **📸写真**: 白いデコジャリを敷き詰めた鉢の上部
        
- [ ] **Day 9：目に見えない栄養**
    
    - **内容**: 実は植物にとって、水と同じくらい「風」が大事。部屋の空気が停滞しないよう、サーキュレーターを活用するのがコツです🍃
        
    - **📸写真**: 風に揺れる葉っぱ
        
- [ ] **Day 10：メンズライクな暮らし**
    
    - **内容**: 男性の部屋にも馴染む、マットな黒。甘くなりすぎない「カッコいい植物のある暮らし」、始めませんか？
        
    - **📸写真**: コンクリート打ちっぱなしやアイアン家具のある部屋と鉢
        
- [ ] **Day 11：見えない機能性**
    
    - **内容**: スーナポットは底穴もしっかり。根腐れを防ぐための機能性も、職人がちゃんと考えて作っています。
        
    - **📸写真**: 鉢の底（裏側）を見せているカット
        
- [ ] **Day 12：会話のきっかけ**
    
    - **内容**: 「これ、元は捨てられる砂だったの？」来客との会話が生まれる植木鉢。ストーリーのあるモノ選びって、なんかいいですよね。
        
    - **📸写真**: 友人と植物を眺めているようなイメージ
        
- [ ] **Day 13：デスクの相棒**
    
    - **内容**: ミニ観葉にぴったりのSサイズ。デスクや洗面所にちょこんと置くだけで、癒やしスポットの完成です🌵
        
    - **📸写真**: デスク周りのPC横にある小さな鉢
        
- [ ] **Day 14：【ケアの基本】**
    
    - **内容**: 受け皿に溜まった水は必ず捨てること！そのままにすると根っこが呼吸できなくなっちゃいます⚠️
        
    - **📸写真**: 受け皿の水を捨てている様子

## 【3週目】ライフスタイル提案

- [ ] **Day 15：夜の表情**
    
    - **内容**: ただの黒じゃない。光が当たると浮かび上がる、砂の粒子。この陰影が、夜の部屋を少しリッチに見せてくれます🌙
        
    - **📸写真**: 夜、間接照明に照らされた鉢
        
- [ ] **Day 16：身近なエコ**
    
    - **内容**: サステナブルって難しく考えなくていい。「次に買う鉢を、リサイクル素材にしてみる」。それだけで立派なエコ活動です🌏
        
    - **📸写真**: 地球や自然を連想させるグリーンの写真
        
- [ ] **Day 17：多肉・サボテン派へ**
    
    - **内容**: 多肉植物やサボテンとも相性抜群。黒い背景が、多肉のぷっくりした形を際立たせてくれますよ。
        
    - **📸写真**: サボテンや多肉の寄せ植え
        
- [ ] **Day 18：お祝いの品に**
    
    - **内容**: 引越し祝いや開店祝いに。「根付く」という意味がある植物ギフト。長く使える良い鉢と一緒に贈りませんか？🎁
        
    - **📸写真**: リボンをかけたラッピング状態の鉢
        
- [ ] **Day 19：使いやすさの秘密**
    
    - **内容**: スーナポットの重さは「軽すぎず、重すぎず」。プラスチックより安定感があり、陶器より扱いやすい。絶妙なバランスです。
        
    - **📸写真**: 女性が片手で鉢を持っている様子
        
- [ ] **Day 20：【飾り方のコツ】**
    
    - **内容**: 1つだけ置くより、高低差をつけて3つ並べるとリズムが出てオシャレに見えます🌿🌿🌿
        
    - **📸写真**: サイズ違いの鉢を3つ並べたバランスの良い写真
        
- [ ] **Day 21：整える時間**
    
    - **内容**: 水をあげて、葉を拭いて。植物の手入れをする時間は、自分の心を整える時間でもありますね。
        
    - **📸写真**: 丁寧に葉を拭いている手元
