# ひろのりのルーティン (Routine Rules)

このファイルは、曜日や頻度に応じて自動的に挿入されるべきタスクを定義します。
Task Architect はこのファイルを参照し、該当するタスクを日誌の「Today's Tasks」に自動追記してください。

### Friday (金曜日)
- [ ] 🌟 週次レビューの実施 (/weekly-review)

### Saturday (土曜日) & Sunday (日曜日)
- (特になし)

## 🔄 Daily Routines (毎日発生するルーティン)

- [ ] 各種SNSや通知のチェックと返信
- [ ] Xポスト・Note・メルマガの作成
- [ ] 砂職人の店: Note・Xポスト・X記事の3点セット作成（New!）

---

## 📂 File Location Rules (保存先ルール)
成果物は必ず以下の指定フォルダに保存すること。

*   **Xポスト**: `04_アウトプット/01_Xポスト/`
*   **Note記事**: `04_アウトプット/03_Note/`
*   **メルマガ**: `04_アウトプット/00_メルマガ/`
*   **Googleドキュメントへの保存**: ユーザーの指示（例：「ドキュメントに保存して」）があった場合のみ実行する。（自動保存はしない）
*   **プロジェクト・その他**: ユーザーが指定したフォルダに保存する。指定がない場合は必ず確認する。
